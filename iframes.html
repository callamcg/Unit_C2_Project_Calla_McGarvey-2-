Iframes

Iframs allow one HTML doc to display the contents of numerous other websites or HTML documents in tables. Using the <iframe> element (shown below), this can be accomplished. However, in order to understand the code behind iframes, users must understand how to make tables in HTML. You can learn how to create tables in the tables page of this website
