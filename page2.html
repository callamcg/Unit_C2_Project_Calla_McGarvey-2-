What you need in a Webpage & How to Start

What do you need?

In your website you usually need to have a head, body, and specific tags. The basic format for a webpage is: 
      <!DOCTYPE html>
       <html lang=”en-US”>
        <head>
         <title> Unit C2 Lesson 2</t2itle>
        </head>
        <body>
         <h1> Heading 1 </h1>
         <p> Write your paragraph here </p>
        </body>
       </html>

Accessability

Use <alt=""> to add a description of an image, allowing screen readers to describe the image.

How to Start?

In order to make a website, you can use a code editor like codepen, github, replit, stackblitz, and more! You can practice code for free on most of these applications and create websites with the code you learn.
