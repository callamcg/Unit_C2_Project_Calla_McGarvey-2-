Creating Tables using HTML

Tables

Tables are helpfult for organize data in a webpage. For example, they can be used to organize scietific data from experiments, show voting results, or put multiple picies of information into rows and coloums. 
 <table> container for all table data
   <tr>
      <th>Name </th>
      <th>Points </th>
   </tr>
   <tr>
      <td>Karel</td>
      <td>31</td>
   </tr>
 </table>

In addition to HTML, a code writer will need a minimal amount of CSS to create boarders around the table, allowing it to be visible in the webpage.
This CSS can be inserted as:
table, th, td {
    border:1px solid black;
  }

Name Points
