Links and Images are necesarry to help your page be interactive and engaging. Links can also help people easily navigate a website with multiple files. Images are a fun way to capture your audience's attention. They also help to break down the content of the page, and often offer another way to understand the topic you are discussing.

Links

      <a href=“pathtoFile”>LinkText</a>
      Absolute or external link:
      <a href=“https://google.com”>Google</a>
      Relative or internal link:
      <a href=“/index.html”>Home Page</a>

Images

       Embedded using <img> element
       Use src attribute to locate the image
       Use alt attribute to describe the image for screen readers
       Ex. Local image
       <img src=“happyface,jpg” alt=”Smiley Face”>
       Remote image
       <img src=“https://ex.com?hf.jpg” alt=”Smiley Face”>
       Sizing Images
       You can control the size of images by using either width or height attributes, or both. The actual dimensions are specified in pixels.
       If you only specify height or width, the image will scale proportionally. If you specify both, the image will fix itself to the dimensions you choose. 
       Ex. 
       <img> src="https://www.thespruce.com/thmb/0k8tYA_KxsO4IMlmiep1KclB2UQ=/750x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/blueberries-vaccinium-spp-3269245-01-42fb9fde200d4c05a2a5d3c6e94a0bf2.jpg" alt="A picture of Blueberries" height=300 width=450></img>

Targets for Links

      Some targets you can use for links are:
      _blank: opens in a new window/tab
      _top: opens at the top of the page
      In your code this can look like: <a href=“pathtoFile” target="_blank">LinkText</a>
